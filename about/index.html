About

30 Days is the design exercise created by me, James Madson. This practice was inspired by my internship with the amazing agency Clearleft in Brighton, England. James Bates, the creative director initiated a 100 days exercise, originating from the most awesome Michael Beirut (Design Observer).

This site is built using Jekyll for architecture, with Susy for layout, Github for storage, Sketch for initial designs, and much more... coming soon.
